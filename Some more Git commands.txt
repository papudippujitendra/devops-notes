 Some more Git commands   
    

git commit -am "file name"  
is used for tracking/adding the data and commit at the same time. This command can be used only for existing files.

git commit -m "file name"
To commit the changes made in working area
  

Changing a commit message :- If a commit message containing unclear incorrect or sensitive information you can amend it locally and push a new commit with a new message to GitHub. you can also change a commit message to add missing information.
